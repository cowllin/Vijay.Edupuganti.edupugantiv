What is ActiveRecord?
	Interface which makes development easier for developer, database clean, part of model in MVC.
What's the difference between a database table, resource and an ActiveRecord model?
	ActiveRecord associates class to database, makes it easy to pull records from database (StackOverFlow), resource model backed by another application's API, database table model backed by it's own database
